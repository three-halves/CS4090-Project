# CS4090-Project
A Event Planning and Scheduling app in the vein of [when2meet](https://www.when2meet.com/?About) created using the Blazor web framework. Currently a work in progress.

## Installation
_This project requires the usage of the .NET runtime, which can be installed [here](https://dotnet.microsoft.com/en-us/download)._

Steps for installing and running this application:
1. Clone this repository
2. `cd` into the cloned directory
3. Run `dotnet run` to launch the application.
4. Access `localhost:4090` in a web browser to access the application.
